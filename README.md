# CppCSInterop
 C++ C# Interop example using WinRT
 
 See blog post https://cppwhispers.blogspot.com/2023/02/cc-interop-for-native-applications.html

# Requirements
- Visual Studio 2022
- CMake
- (Windows SDK 10.0.22621.0)

# Instructions
    setup.bat

# Build
    cmake --build build

# Run
    cd distrib
    CppTextEXE.exe
